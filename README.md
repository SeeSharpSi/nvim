# nvim-config
This is my nvim config. Enjoy!
## Dependencies
* [Packer](https://github.com/wbthomason/packer.nvim)
## Install
Go into lua/theprimeagen/packer and run these vim commands
```vim
:so
:PackerSync
```
Use ```:lua ColorMyPencils()``` for transparent background
